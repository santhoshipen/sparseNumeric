# Mean of a sparse_numeric vector

Computes the mean of the full vector without expanding to dense form.

## Usage

``` r
# S4 method for class 'sparse_numeric'
mean(x, ...)
```

## Arguments

- x:

  A sparse_numeric vector.

- ...:

  Ignored, included for consistency with the generic.

## Value

A numeric scalar, the mean of the full vector.
